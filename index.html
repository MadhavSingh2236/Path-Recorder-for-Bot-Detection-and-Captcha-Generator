Quick instructions

This is a simple tool to record mouse movements and create a JSON object from them.

  • Start by clicking on the big rectangle and you'll see a recording icon appear bottom left. Any mouse movement now adds to the dataset. Click again to end the recording.
  • Enable "Limit Points" to record fewer points and get a smaller dataset.
  • Once recorded, you can re-play the paths with the play button.
  • Edit the dataset in the text box below and save it as a file by clicking the download link.
  • The load button allows you to load datasets.
  • If you load an image, it will become the background to allow you to trace it.
  • Click the bin to delete all your work and start over.
